# Face-Recognition-Access-System
Face Recognition Access System

#### Instructions
* Process sequence : Recongnize > Authenticate > Unlock
* Recognize : OpenCV-Based detection.
* Authenticate : RSID-Based authentication.
* Unlock : Servo(PWM)-Based switch press
